Hướng Dẫn Sử Dụng

(Bản cập nhật ngày 18.1.2024: bản đã fix chấm điểm, crash khi chuyển tình huống)

  1. Bước 1: Download bộ cài đặt phiên bản tương ứng với Hệ điều hành Windows 64-bit hoặc Windows 32-bit theo đường link trên.
  2. Bước 2: Giải nén bộ cài, mật khẩu giải nén mophonggiaothong2024
  3. Bước 3: Chạy file setup.exe và thực hiện tuần tự từng bước để cài đặt

Yêu Cầu Hệ Thống

  • CPU: Intel Core i3 thế hệ 2 trở lên
  • RAM: 4GB trở lên
  • Dung lượng ổ cứng: Trống ít nhất 10GB

Kiểm Tra Máy Tính

  1. Bước 1: Mở Control Panel và chọn System and Security.
  2. Bước 2: Chọn System để xem thông tin về hệ điều hành và cấu hình máy tính.
  3. Bước 3: Kiểm tra mục System type để biết máy tính của bạn là 32-bit hay 64-bit.
  4. Bước 4: Kiểm tra mục Processor và Installed memory (RAM) để biết cấu hình CPU và RAM.
